4.1.E.1A Deceleration Time and Distance, Manual Wheel Brakes, Dry Runway, No Reverse Thrust - Heavy

OBJECTIVE

Demonstrate that the simulator deceleration time and distance, manual wheel brakes, dry runway, no reverse thrust is compliant with FAA Part 60 (Fixed Wing FFS) Change 2, CS-FSTD(A) Issue 2, ICAO Doc 9625 4th Edition, CCAR AC-60-FS, (UK) CS-FSTD(A) Issue 2 standards with respect to the reference data.

PROCEDURE

Select and run the case using TestDrive from the IOS. For information on how to use TestDrive, refer to Section 2 of this document.

  1. Automated - ensure that TestDrive is set up to run the case in auto mode.
  2. Manual - ensure that TestDrive is set up to run the case in manual mode. An initial conditions checklist and a list of manual procedures are provided on the following

EVALUATION

A time history of the simulator deceleration performance will be produced along with a table summarizing the stopping time and distance and comparing it with flight test data. An initial conditions checklist is available to verify the initial setup for each case. An out-of-tolerance condition will be flagged in the table by an asterisk (* ) next to the simulator generated

TOLERANCES

+1.5s or +5% of time
+200ft (61 m) or =10% of distance for dist. up to 4000ft (1220m), whichever is smaller +5% of distance for distance greater than 4000ft (1220m)
